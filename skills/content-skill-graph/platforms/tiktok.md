# TikTok — Raw, 45-60 Second Scripts

## Role
The wake-up call. Fast, direct, slightly aggressive truth.

## Audience
- Scrolling fast, thumb ready to swipe
- Younger builders, founders
- People who learn by watching, not reading

## DNA
- **Angle:** Urgent reality check
- **Hook:** First 2 seconds = pattern interrupt
- **Tone:** Raw, conversational, no polish
- **Format:** 45-60 second vertical video script
- **Depth:** One point, hammered home

## Rules

### Must Have
- Hook in first 2 seconds (visual + verbal)
- Pattern interrupt (unexpected statement/visual)
- 1-3 key points max
- Clear CTA (follow, comment, save)

### Never
- Slow intro ("Hey guys, today I want to...")
- Corporate polish or scripted energy
- More than 3 main points
- Apologetic tone

## Script Structure

```
0-2 sec:  Hook (pattern interrupt)
2-10 sec: Context (the problem, fast)
10-30 sec: Core point (evidence/example)
30-45 sec: Takeaway (what to do)
45-60 sec: CTA + loop bait
```

## Hook Formulas (First 2 Seconds)

### 1. The Callout
> "Stop doing [common thing]. It's killing your [outcome]."

### 2. The Question
> "Why does your [thing] keep failing? Here's why."

### 3. The Contrarian
> "Everyone says [popular opinion]. They're wrong."

### 4. The Number
> "[Specific stat]. That's your problem."

## Visual Directions

- **Framing:** Close-up, eye contact with camera
- **Background:** Real workspace (not staged)
- **Energy:** Tired but direct, not hype
- **Text overlays:** Key phrases only (3-5 words max)
- **Cuts:** Every 3-5 seconds to maintain attention

## Example Script

**Topic:** AI Governance Gap

```
[0-2s]  "Your prompts aren't versioned. That's a lawsuit waiting."
        [Text overlay: "PROMPTS = UNVERSIONED?"]

[2-10s] "You git commit your code. CI/CD pipelines. Full audit trail.
         But your prompts? Slack threads and hope."

[10-30s] "EU AI Act drops next year. They don't want your intentions.
         They want decision traceability. Who changed what prompt and why?
         If you can't show that, you're exposed."

[30-45s] "Three things: Map where prompts touch data. Version them like code.
         Build approval gates. Same rigor as your production deployments."

[45-60s] "Stop treating prompts like chat messages. They're infrastructure.
         Fix the gap before someone asks you to explain it in a deposition.
         Follow for more truth bombs."
```

## Caption Formula

```
Hook (first line, no "..." truncation)
↓
1-2 sentences of context
↓
CTA (Follow, Comment your take)
↓
3-5 hashtags (#AI, #Engineering, #Startup, #Compliance)
```

## Repurposing Notes

| From Blog | TikTok Transformation |
|-----------|----------------------|
| 2,000 words | 60 seconds, one point |
| Nuanced argument | Direct callout |
| "On one hand..." | Pick a side, no hedging |
| Comprehensive guide | Top 1-3 actions only |

## Validation Checklist
- [ ] Hook in first 2 seconds?
- [ ] Pattern interrupt (visual or verbal)?
- [ ] Under 60 seconds when read aloud?
- [ ] 3 or fewer main points?
- [ ] Clear CTA at end?
- [ ] Sounds like a human, not a script?
- [ ] Text overlays planned for key phrases?

## Platform-Specific Voice
Same tired engineer. But now they're leaning into the camera at 11pm, had enough coffee, and they're telling you the thing you need to hear. Raw, unpolished, real.

## Posting Strategy
- **Frequency:** 3-5x per week
- **Best times:** 7-9am, 12-1pm, 7-9pm (local audience time)
- **Engagement:** Reply to comments within 2 hours (boosts algorithm)

## Tools
- CapCut (editing, captions)
- Teleprompter apps (if needed, but natural is better)
- Phone camera (no need for pro gear — raw wins)
